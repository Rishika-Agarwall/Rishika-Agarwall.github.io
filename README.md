# Rishika Agarwal | Professional Portfolio Website

A formal, modern, and fully responsive single-page portfolio website designed for **Rishika Agarwal**, MSc Computer Science student at the University of Freiburg.

## 🔗 Live Site
* **Live Deployment:** [rishika-agarwal.surge.sh](http://rishika-agarwal.surge.sh/)

## 🛠️ Built With
* **Structure:** Semantic HTML5
* **Styling:** Vanilla CSS3 (Custom HSL Color Tokens, Flexbox, CSS Grid)
* **Interactivity:** Vanilla Javascript (ES6)
* **Icons:** FontAwesome v6

## ✨ Key Features
1. **Interactive Theme Switcher:** Fully customizable Light/Dark mode switcher that automatically matches user system preference and persists settings via `localStorage`.
2. **Dynamic Scroll Indicator:** Sidebar menu navigation dynamically highlights the current active section as the user scrolls.
3. **Research Log & Blog:** 
   * A clean, collapsible entry form to log new research observations or literature reviews.
   * Real-time saving of posts using client-side `localStorage`.
   * Secure URL parsing (auto-linkification) that renders plain URL strings inside post content as clickable links.
4. **Professional Resume Integration:** Includes a link to view/download the compiled PDF version of the formal CV.

## 📁 Repository Structure
* `index.html` - Core layout and markup
* `style.css` - Responsive styling and theme tokens
* `script.js` - Dynamic scroll nav, theme toggle, and local storage blog logic
* `avatar.svg` - Minimal vector silhouette avatar placeholder
* `cv.pdf` - High-quality formal curriculum vitae
